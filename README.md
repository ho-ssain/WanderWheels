# WanderWheels

WanderWheels is a Car rental System provides a convenient, on-demand solution for short-term vehicle access

[WanderWheels Live Site](https://ho-ssain.github.io/WanderWheels/)

## User Role

1. Customer
2. Car Owner

## Features

1. cars by Owner-id
2. cars by location
3. search car by location
4. add new car
5. access
6. create booking
